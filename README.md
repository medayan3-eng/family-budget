# family-budget
# 💰 תוכנית חיסכון משפחתית

אפליקציית Streamlit לניהול תקציב, מעקב השקעות וחיסכון משפחתי.

## 🚀 התקנה מהירה

```bash
# שכפול הפרויקט
git clone https://github.com/YOUR_USERNAME/family-budget.git
cd family-budget

# התקנת dependencies
pip install -r requirements.txt

# הפעלה
streamlit run app.py
```

## 📁 מבנה הפרויקט

```
family-budget/
├── app.py                  # לוח בקרה ראשי
├── pages/
│   ├── 1_Investments.py   # ניהול השקעות ומניות
│   ├── 2_Income.py        # מעקב הכנסות
│   ├── 3_Expenses.py      # מעקב הוצאות
│   └── 4_Analysis.py      # ניתוח וגרפים
├── utils/
│   ├── data_manager.py    # ניהול נתונים (JSON)
│   ├── price_fetcher.py   # שליפת מחירים
│   └── constants.py       # קטגוריות
├── data/                  # נתוני המשתמש (לא נכנס לגיט!)
└── requirements.txt
```

## 📊 פיצ'רים עיקריים

### לוח בקרה
- שווי תיק השקעות כולל
- הכנסות והוצאות חודשיות
- חיסכון חודשי ושיעור חיסכון
- גרפים: הרכב תיק + הוצאות לפי קטגוריה
- היסטוריית הכנסות/הוצאות

### השקעות
- **מניות אמריקאיות** — yfinance (AAPL, SPY, QQQ...)
- **מניות ישראליות (TASE)** — yfinance עם סיומת .TA
- **קרנות נאמנות ישראליות** — TASE API לפי מספר קרן
- **נכסים ידניים** — עדכון מחיר ידני
- המרת USD → ILS אוטומטית
- **עדכון מחירים פעם ביום** (cache יומי)

### הכנסות
- הוספה לפי חודש
- כפתורי הוספה מהירה (משכורת קבועה, דמי לידה)
- קטגוריות: משכורת, דמי לידה, בונוס, הכנסה פסיבית

### הוצאות
- **קטגוריות ראשיות** עם תת-קטגוריות:
  - 🏠 דיור (שכר דירה, ארנונה, חשמל, מים, גז...)
  - 🛒 מזון
  - 👶 תינוקת
  - 🚗 תחבורה
  - 👕 ביגוד
  - 💊 בריאות
  - 🎬 פנאי
  - ועוד...
- פיצול לפי קטגוריה (עוגה + בר)
- הוספה מהירה להוצאות קבועות

### ניתוח
- גרף הכנסות/הוצאות/חיסכון לאורך זמן
- שיעור חיסכון עם יעד 20%
- ניתוח הוצאות לפי קטגוריה
- תובנות אוטומטיות
- ייצוא נתונים CSV

## 🔧 הגדרת קרנות ישראליות

להוספת קרן נאמנות ישראלית:
1. מצא את מספר הקרן (7 ספרות) באתר TASE או בפלטפורמת הברוקר שלך
2. בדף "השקעות" בחר "קרן נאמנות ישראלית"
3. הזן את מספר הקרן
4. אם המחיר לא נשלף אוטומטית, השתמש ב"עדכון מחיר ידני"

### טיקרים נפוצים

| נכס | טיקר/מספר |
|-----|-----------|
| S&P500 (iShares) | IVV |
| Nasdaq ETF | QQQ |
| תל אביב 125 | TA125.TA |
| תל בונד | TELBND.TA |
| שקל/דולר | USDILS=X |

## 🔒 פרטיות
